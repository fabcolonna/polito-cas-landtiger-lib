LE = Latch Enable (20)

DIR = Bus Direction (21) (0 = Write, 1 = Read)

CS = Chip Select (22)

RS = Register Select (23) (0 = Command, 1 = Data)

WR = Write Operation (24)

RD = Read Operation (25)
